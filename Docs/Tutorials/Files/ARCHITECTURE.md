# Files tutorial

Reading/ writing from the file system on the watch

* In this tutorial we will show a user how
making settings files/ reading settings files
* In step 1 - The user will save a JSON file with
default values for:
{“Activtiy type”:”hot air ballooning”,
"heartRateZones":[115,134,153,172]}
* In step 2 the User will read this data out to
the debug port.
* The user is then encouraged to edit this file
and read out the new settings in the debug
line.
* If the file is not detected it will be re-created
with the default values
