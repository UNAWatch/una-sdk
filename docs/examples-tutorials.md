# Examples & Tutorials

This section provides practical, real-world examples and step-by-step tutorials to help you master Una-Watch development.

## Beginner Tutorials

### 1. Digital Clock Face
Learn how to build a basic digital clock face that updates every minute.
- **Concepts**: Time API, basic TouchGFX text labels.
- **Difficulty**: ⭐

### 2. Step Counter Glance
Build a simple glance app that displays the current step count.
- **Concepts**: Sensor API (Step Counter), Glance UI (240x60).
- **Difficulty**: ⭐⭐

## Intermediate Tutorials

### 3. Heart Rate Monitor with History
Create an app that monitors heart rate and displays a graph of the last hour.
- **Concepts**: Sensor API (PPG), File System (Data logging), Custom TouchGFX widgets.
- **Difficulty**: ⭐⭐⭐

### 4. Weather Utility
A utility app that fetches weather data from a companion phone via BLE.
- **Concepts**: BLE Communication, JSON parsing, App lifecycle.
- **Difficulty**: ⭐⭐⭐

## Advanced Tutorials

### 5. Multi-Sensor Fusion Tracker
An activity tracker that combines accelerometer and heart rate data for precise calorie calculation.
- **Concepts**: Advanced Sensor Layer, Math libraries, Power optimization.
- **Difficulty**: ⭐⭐⭐⭐⭐
