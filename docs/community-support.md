# Community & Support

Join the Una-Watch developer community to share your apps, get help, and stay updated on the latest SDK features.

## Developer Resources

- **GitHub**: [Report bugs and contribute to the SDK](https://github.com/UNAWatch/una-sdk)

## Getting Help

- Search the guide and github repository.
- Check the SDK/Examples to see if a similar feature is implemented.
- Ask a question on the GitHub.
- For critical product issues, contact our support team at `support@una-watch.dev`.

## Contributing

We welcome contributions to the Una-Watch SDK!

- **Documentation**: Submit a PR to improve these docs.
- **Samples**: Share your app templates with the community.
- **Core SDK**: Help us improve the kernel interfaces and build tools.

---

*Thank you for being part of the Una-Watch ecosystem!*
